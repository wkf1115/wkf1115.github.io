Redis的安装

2020-07-10 1 min read

docker下的安装：

  • 1.docker search redis
  • 2.docker pull redis:latest
  • 3.docker run -itd --name redis-test -p 6379:6379 redis
  • 4.docker exec -it redis-test /bin/bash
  • 5.接下来运行redis-cli可以进入redis的客户端。
